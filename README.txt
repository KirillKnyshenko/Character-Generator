Нужно создать кастомизацию персонажа.
По кнопке Generate должен создаваться персонаж с случайными частями тела(все исходные части использовать не нужно, главное - сам принцип).
По кнопке Play игра должна переходить на сцену Game где должен быть отображен сгенерированный персонаж.
Важным требованием должно быть отделение логики кастомизации от unity ui.
Вся логика должна быть в классах, наследованных не от MonoBehaviour.
